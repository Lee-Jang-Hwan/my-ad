## 🎯 사업 개요

### 비전
**"모든 소상공인이 쉽고 빠르게 전문적인 홍보영상을 만들 수 있는 세상"**

### 미션
AI 기술을 활용하여 마케팅 전문 지식이 없는 소상공인도 클릭 몇 번만으로 SNS용 홍보영상을 제작할 수 있는 플랫폼을 제공합니다.

### 핵심 가치 제안 (Value Proposition)

1. **초간편 제작**: 이미지 1장 + 상품명만으로 영상 완성
2. **전문 품질**: AI가 광고 문구, 영상 편집, TTS, 자막까지 자동 생성
3. **빠른 속도**: 5분 이내 완성 (기존 1-2시간 → 5분)
4. **저렴한 비용**: 영상 제작 비용 1/10 절감 (예상)
5. **즉시 활용**: SNS(Instagram, Facebook, YouTube) 직접 업로드 지원

### 사업 목표

#### 단기 목표 (3개월)
- MVP 출시 및 베타 테스트
- 초기 사용자 100명 확보
- 영상 생성 성공률 80% 달성
- 평균 생성 시간 5분 이내 달성

#### 중기 목표 (6개월)
- 유료 전환율 10% 달성
- 월간 활성 사용자(MAU) 500명
- 영상 생성 누적 1,000개 돌파
- SNS 자동 업로드 기능 완성

#### 장기 목표 (1년)
- 월 구독자 1,000명 달성
- 월 매출 $10,000 달성
- B2B 솔루션 출시 (대행사, 프랜차이즈)
- 해외 진출 (영어, 일본어 지원)

## 📊 시장 분석

### 시장 규모 (TAM/SAM/SOM)

#### TAM (Total Addressable Market)
- **글로벌 동영상 마케팅 시장**: $135B (2024년 기준)
- **소상공인 수**: 전 세계 약 4억 개
- 잠재 시장 규모: 매우 크고 빠르게 성장 중

#### SAM (Serviceable Addressable Market)
- **한국 소상공인 수**: 약 650만 명
- SNS 마케팅 필요성 인식: 약 40% (260만 명)
- 영상 마케팅 관심층: 약 30% (78만 명)

#### SOM (Serviceable Obtainable Market)
- **1년 차 목표 고객**: 1,000명 (0.13% 점유율)
- **3년 차 목표 고객**: 10,000명 (1.3% 점유율)
- 실현 가능한 목표 시장

### 타겟 고객 세분화

#### Primary Target (1차 타겟)
**소상공인 (30-50대, 월 마케팅 예산 10-50만원)**

- **Pain Points**:
  - 영상 제작 비용 부담 (1건당 30-100만원)
  - 전문 지식 부족 (편집 프로그램 어려움)
  - 시간 부족 (영업에 집중해야 함)
  - SNS 마케팅 효과는 알지만 실행 못함

- **Gains**:
  - 저렴한 비용으로 전문 영상 제작
  - 클릭 몇 번으로 완성
  - 즉시 SNS 업로드 가능
  - 매출 증대 기대

- **업종**:
  - 카페/음식점 (메뉴 홍보)
  - 뷰티/네일샵 (시술 홍보)
  - 패션/액세서리 (상품 홍보)
  - 헬스/필라테스 (회원 모집)
  - 반려동물 용품 (상품 홍보)

#### Secondary Target (2차 타겟)
**1인 크리에이터, 스마트스토어 판매자**

- SNS 콘텐츠 제작 수요 많음
- 빠른 제작 속도 필요
- 비용 민감도 높음

#### Tertiary Target (3차 타겟, 추후)
**마케팅 대행사, 프랜차이즈 본사**

- 대량 영상 제작 수요
- B2B 맞춤형 솔루션
- 높은 객단가 가능

### 시장 트렌드

1. **숏폼 영상 급증**
   - 릴스, 쇼츠, 틱톡 등 짧은 영상 선호도 증가
   - 평균 시청 시간 8-15초에 최적화

2. **AI 생성 콘텐츠 수용도 증가**
   - ChatGPT, Midjourney 등으로 AI 친숙도 높아짐
   - AI 생성 콘텐츠에 대한 거부감 감소

3. **No-Code/Low-Code 도구 수요**
   - Canva, Notion 등 전문 지식 없이 사용 가능한 도구 인기
   - "쉬움"이 가장 중요한 구매 요인

4. **영상 마케팅 효과 증명**
   - 영상 콘텐츠의 전환율이 텍스트/이미지 대비 2-3배 높음
   - 소상공인들도 인지하고 있으나 실행력 부족

## 🎨 서비스 설명

### 서비스 개요
**"이미지 1장 + 상품명 입력 → 5분 후 SNS용 홍보영상 완성"**

AI가 광고 카피 작성, 이미지 보정, 영상 제작, 음성 합성, 자막 생성, 최종 편집까지 모두 자동으로 처리합니다.

### 핵심 기능 흐름

```
[사용자 입력]
   ↓
이미지 업로드 (드래그 앤 드롭)
   ↓
상품명 입력 (예: "시그니처 아메리카노")
   ↓
"영상 생성 시작" 버튼 클릭
   ↓
[AI 자동 처리 - 약 5분]
   ↓
1단계: 광고 카피 생성 (Gemini AI)
   "아침을 깨우는 진한 아로마, 시그니처 아메리카노"
   ↓
2단계: 이미지 정제 (Gemini Vision)
   밝기, 대비, 색감 자동 보정
   ↓
3단계: 영상 생성 (Veo 3.1)
   이미지 기반 8-10초 동영상 생성
   ↓
4단계: TTS 생성 (Google TTS)
   광고 카피를 자연스러운 음성으로 변환
   ↓
5단계: 자막 생성 (SRT Generator)
   음성에 맞춰 자막 타이밍 자동 설정
   ↓
6단계: 최종 합성 (FFmpeg)
   영상 + 음성 + 자막 결합
   ↓
[완성]
   ↓
영상 미리보기 및 다운로드
   ↓
SNS 직접 업로드 (Instagram, Facebook, YouTube)
```

### 사용자 경험 (UX)

#### 1. 홈페이지
- 샘플 영상 자동 재생 (실제 생성된 영상)
- "지금 무료로 시작하기" CTA 버튼
- 사용 방법 3단계 설명

#### 2. 업로드 페이지
- 직관적인 드래그 앤 드롭
- 이미지 프리뷰 즉시 표시
- 간단한 상품명 입력 폼
- 예상 소요 시간 안내 (약 5분)

#### 3. 진행 상태 페이지
- 실시간 진행 단계 표시 (1/7 → 7/7)
- 각 단계별 로딩 애니메이션
- 남은 예상 시간 카운트다운
- 중간에 나가도 됨 (백그라운드 처리)

#### 4. 완성 페이지
- 영상 플레이어 (재생/일시정지/전체화면)
- 다운로드 버튼 (MP4 형식)
- SNS 공유 버튼 (Instagram, Facebook, YouTube)
- 링크 복사 (공유용)

#### 5. 대시보드
- 생성한 영상 목록 (썸네일 그리드)
- 상태별 필터 (생성 중, 완료, 실패)
- 날짜 정렬 (최신순, 오래된순)

## 🛠️ 기술 스택

### Frontend
- **Next.js 15.5.6** (React 19, App Router)
- **TypeScript** (strict mode)
- **Tailwind CSS v4** + shadcn/ui
- **Clerk** (인증, 한국어 지원)
- **Supabase Realtime** (진행 상태 실시간 업데이트)

### Backend
- **Supabase** (PostgreSQL + Storage)
- **Server Actions** (데이터 조작)
- **n8n** (워크플로우 엔진)

### AI & Video Processing
- **Google Vertex AI**:
  - Gemini 2.5 Flash (광고 카피 생성)
  - Gemini Vision (이미지 정제)
  - Veo 3.1 (영상 생성)
  - Google TTS (음성 합성)
- **FFmpeg** (영상 합성)

### Infrastructure
- **Vercel** (Frontend 호스팅)
- **Supabase Cloud** (Database + Storage)
- **n8n Cloud** (워크플로우 실행, 추후)

### 기술적 차별점
1. **No-Code AI 워크플로우**: n8n으로 비개발자도 워크플로우 수정 가능
2. **Realtime 피드백**: Supabase Realtime으로 진행 상태 즉시 확인
3. **Clerk 인증**: 간편한 소셜 로그인 (구글, 카카오)
4. **Server Actions**: 빠른 응답 속도 및 보안

## 🚀 MVP 기능

- 단계별 로딩 애니메이션
- 예상 소요 시간 표시

**6. 영상 관리**

- 상태별 필터 (생성 중, 완료, 실패)
- 영상 상세 페이지

**7. 다운로드 기능**
- MP4 다운로드
- 파일명 자동 생성 (상품명 + 날짜)

**8. 공유 기능**
- 링크 복사 (클립보드 API)

**9. UI/UX**
- 다크모드 지원
- 반응형 디자인 (모바일/태블릿/데스크톱)
- 로딩 스켈레톤
- 접근성 (ARIA 라벨, 키보드 내비게이션)

- 타겟팅: 30-50대, 소상공인, 카페/음식점 운영자
- 크리에이티브: 실제 생성 영상 before/after
- 예산: 월 $500

**네이버 광고**
- 검색 광고: "홍보영상", "AI 영상 제작"
- 디스플레이 광고: 블로그/카페
- 예산: 월 $300

#### 3. 파트너십 & 제휴

**소상공인 지원 기관**

- 예산: 월 $200 (기프트 카드 등)

**마케팅 대행사**
- 영상 제작 필요한 대행사에 제휴 제안
- 수수료 공유 (20%)
- Win-Win 구조

#### 4. 추천인 프로그램

**양방향 보상**
- 추천인: 피추천인이 유료 전환 시 1개월 무료
- 피추천인: 가입 시 첫 달 50% 할인
- 바이럴 효과 극대화

**Viral Loop 설계**
- 생성 영상에 "이 영상은 [서비스명]에서 제작되었습니다" 워터마크
- SNS 공유 시 자동으로 서비스 홍보
- 무료 사용자만 워터마크 (유료는 제거)

### 성장 해킹

**온보딩 최적화**

## ⚠️ 위험 요소 및 대응

### 기술적 위험

#### 1. AI 생성 품질 불안정
**위험**: Veo 3.1 영상 품질이 기대에 못 미칠 수 있음

**대응**:
- 대체 AI 모델 준비 (Runway Gen-3, Pika)
- 사용자 피드백 기반 지속 개선
- 품질 낮은 영상 재생성 무료 제공

#### 2. API 비용 급증
**위험**: 사용자 증가 시 Google Vertex AI 비용 폭증

**대응**:
- 사용량 모니터링 및 알림 시스템
- 배치 처리로 비용 최적화
- 자체 AI 모델 학습 고려 (장기)

#### 3. 서비스 안정성
**위험**: n8n 워크플로우 중단 시 전체 서비스 마비

**대응**:
- n8n Cloud로 이전 (SLA 보장)
- 자동 재시도 로직
- 에러 모니터링 (Sentry)
- 장애 발생 시 사용자 알림

### 비즈니스 위험

#### 4. 낮은 유료 전환율
**위험**: 무료 사용자는 많지만 유료 전환 안 됨

**대응**:
- 무료 플랜 제한 강화 (월 3개 → 2개)
- 유료 플랜 가치 강조 (워터마크 제거, SNS 업로드)
- 시간 제한 프로모션 (첫 달 50% 할인)

#### 5. 고객 이탈 (Churn)
**위험**: 1-2개월 사용 후 구독 취소

**대응**:
- 사용 빈도 모니터링 (1개월 미사용 시 알림)
- 이메일 마케팅 (새 기능, 성공 사례)
- 연간 결제 유도 (20% 할인)
- 취소 시 피드백 수집 후 개선

#### 6. 경쟁사 모방
**위험**: 대형 플랫폼(Canva 등)이 유사 기능 추가

**대응**:
- 빠른 제품 개선 (First Mover Advantage)
- 한국 시장 특화 (TTS 품질, 한국어 UI)
- 커뮤니티 형성 (충성 고객 확보)
- 특허/상표 등록 고려

### 법적/규제 위험

#### 7. AI 생성 콘텐츠 저작권
**위험**: AI 생성 영상의 저작권 귀속 불명확

**대응**:
- 약관에 명시 (사용자가 영상 저작권 보유)
- 법률 자문 확보
- 글로벌 AI 저작권 동향 지속 모니터링

#### 8. 개인정보 보호
**위험**: GDPR, 개인정보보호법 위반

**대응**:
- Clerk 사용 (GDPR 준수)
- 개인정보 처리방침 명확히
- 데이터 최소 수집 원칙
- 정기 보안 감사

### 시장 위험

#### 9. 시장 수요 부족
**위험**: 소상공인들이 영상 마케팅에 관심 없을 수 있음

**대응**:
- MVP 빠르게 출시 (시장 검증)

**위험**: 경제 불황 시 마케팅 예산 삭감

**대응**:
