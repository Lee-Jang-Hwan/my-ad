# 삽가능 스튜디오 시연 영상 제작 가이드

## 영상 개요

| 항목 | 내용 |
|------|------|
| **총 길이** | 5분 (300초) |
| **녹화 도구** | OBS Studio |
| **TTS** | 타입캐스트 |
| **편집** | 캡컷 (CapCut) |
| **해상도** | 1920x1080 (Full HD) |
| **프레임** | 30fps |

---

## 사전 준비사항

### OBS 설정
- [ ] 브라우저 소스: Chrome (1920x1080)
- [ ] 마우스 클릭 효과 활성화 (선택)
- [ ] 불필요한 북마크바, 확장프로그램 숨기기
- [ ] 다크모드로 통일 (시각적 일관성)

### 테스트 데이터 준비
- [ ] 상품 이미지 2-3개 (깔끔한 배경, 고화질)
- [ ] 테스트 계정 로그인 상태
- [ ] 크레딧 충분히 보유 (시연용)
- [ ] 이미 생성된 샘플 영상/이미지 몇 개 (대시보드용)

### 타입캐스트 설정
- [ ] 목소리: 전문적이고 친근한 남성/여성 음성
- [ ] 속도: 1.0x (자연스러운 속도)
- [ ] 감정: 밝고 자신감 있는 톤

## 타임라인 상세 계획

### 1. 인트로 (0:00 - 0:20) [20초]

#### 화면 연출
```
[0:00-0:05] 로고 애니메이션 또는 타이틀 카드
            "삽가능 스튜디오"
            부제: "AI 광고 영상 & 이미지 자동 생성"

[0:05-0:20] 홈페이지 히어로 섹션 천천히 스크롤
            샘플 영상들이 재생되는 모습
```

#### 나레이션 스크립트
```
"상품 사진 한 장만 있으면, 광고 영상이 자동으로 만들어진다면 어떨까요?
삽가능 스튜디오를 소개합니다."
```

#### 캡컷 편집 팁
- 인트로 타이틀: 페이드 인 효과
- 배경음악: 밝고 경쾌한 BGM 시작 (볼륨 20-30%)
- 자막: 중앙 하단, 흰색 텍스트 + 검은 외곽선

### 2. 문제 제기 & 솔루션 (0:20 - 0:50) [30초]

#### 화면 연출
```
[0:20-0:35] 홈페이지 "How It Works" 섹션 보여주기
            3단계 프로세스 아이콘 강조

[0:35-0:50] 샘플 영상 갤러리 스크롤
            실제 생성된 광고 영상 클릭해서 재생
```

#### 나레이션 스크립트
```
"기존에 광고 영상을 만들려면, 촬영, 편집, 디자인까지
전문가와 많은 시간이 필요했습니다.

삽가능 스튜디오는 단 3단계로 해결합니다.
상품 사진 업로드, 광고 문구 선택, 그리고 AI가 영상을 생성합니다.

실제로 어떻게 작동하는지 보여드리겠습니다."
```

#### 캡컷 편집 팁
- "3단계" 강조할 때 텍스트 팝업 효과
- 숫자 카운트 애니메이션 (1 → 2 → 3)
- 전환: 부드러운 크로스 디졸브

### 4. 비디오 생성 시연 (1:10 - 2:30) [80초] ⭐ 핵심

#### 화면 연출
```
[1:10-1:20] 사이드바에서 "영상 생성" 메뉴 클릭
            업로드 폼 페이지 진입

[1:20-1:35] 상품 이미지 드래그 앤 드롭
            이미지 미리보기 표시
            상품명 입력: "프리미엄 무선 이어폰"

[1:35-1:45] "광고 문구 생성" 버튼 클릭
            로딩 스피너 표시
            6개의 광고 문구 카드 생성됨

[1:45-2:00] 각 광고 문구 카드 훑어보기
            마음에 드는 문구 선택 (클릭)
            "이 문구로 영상 생성" 버튼 클릭

[2:00-2:20] 생성 진행률 페이지
            실시간 진행률 바 표시
            각 단계 아이콘 변화 (분석 → 생성 → 렌더링)
            (시간 절약을 위해 빠르게 편집)

[2:20-2:30] 생성 완료!
            영상 플레이어에서 재생
            "다운로드" 버튼 보여주기
```

#### 나레이션 스크립트
```
"영상 생성 메뉴로 이동합니다.

상품 이미지를 드래그해서 업로드하고,
상품명을 입력합니다. 프리미엄 무선 이어폰이라고 해볼게요.

이제 AI가 광고 문구를 생성합니다.
잠시만 기다리면... 6개의 광고 문구가 생성되었습니다.

각 문구를 살펴보고, 마음에 드는 걸 선택합니다.
이 문구가 좋네요. 선택하고 영상 생성을 시작합니다.

AI가 실시간으로 영상을 만들고 있습니다.
분석, 이미지 생성, 영상 렌더링 순서로 진행됩니다.

완료되었습니다! 바로 확인해볼까요?
이렇게 전문적인 광고 영상이 자동으로 만들어졌습니다.
다운로드해서 바로 사용할 수 있습니다."
```

#### 캡컷 편집 팁
- 업로드 순간: 드래그 모션 강조 효과
- 광고 문구 생성 중: 로딩 부분 2배속
- 진행률 화면: 타임랩스로 빠르게 (실제 시간 단축)
- 완성 영상 재생: 원본 속도, 소리 포함
- 자막: 주요 버튼명 강조 ("광고 문구 생성", "영상 생성")

### 6. 스토리보드 시연 (3:10 - 4:10) [60초] ⭐ 차별화 기능

#### 화면 연출
```
[3:10-3:20] 사이드바에서 "스토리보드" 메뉴 클릭
            "새 스토리보드" 버튼 클릭
            제목 입력: "신제품 런칭 영상"

[3:20-3:35] 스토리보드 설정 화면
            - 화면 비율 선택 (16:9)
            - 씬 길이 선택 (8초)
            "AI 자동 생성" 버튼 클릭

[3:35-3:50] AI가 씬 구조 자동 생성
            5개 씬이 타임라인에 표시됨
            각 씬의 설명 텍스트 확인

[3:50-4:00] 씬 하나 클릭해서 상세 설정 보여주기
            - 카메라 각도 선택
            - 전환 효과 선택
            - 대사/나레이션 입력

[4:00-4:10] 전체 스토리보드 미리보기
            또는 이미 생성된 스토리보드 영상 보여주기
```

#### 나레이션 스크립트
```
"더 정교한 영상이 필요하다면, 스토리보드 기능을 사용해보세요.

새 스토리보드를 만들고, 기본 설정을 선택합니다.
AI 자동 생성 버튼을 누르면...

AI가 스토리 구조를 자동으로 만들어줍니다.
인트로, 제품 소개, 기능 설명, 클로징까지 5개 씬이 생성되었네요.

각 씬을 클릭하면 카메라 각도, 전환 효과, 나레이션까지
세부적으로 조정할 수 있습니다.

이렇게 전문가 수준의 스토리보드 기반 영상을
누구나 쉽게 만들 수 있습니다."
```

#### 캡컷 편집 팁
- 스토리보드 타임라인: 화살표로 씬 순서 강조
- AI 생성 과정: 반짝이는 효과
- 설정 화면: 주요 옵션에 박스 하이라이트
- "전문가 수준" 자막 강조

## 캡컷 편집 체크리스트

### 자막 스타일
- **폰트**: 프리텐다드 또는 노토 산스 (깔끔한 고딕체)
- **크기**: 48-56px
- **색상**: 흰색 (#FFFFFF)
- **외곽선**: 검은색 2px
- **위치**: 화면 하단 중앙 (안전 영역 내)

### 효과 사용
| 상황 | 효과 |
|------|------|
| 화면 전환 | 크로스 디졸브 (0.3초) |
| 중요 버튼 클릭 | 원형 하이라이트 + 확대 |
| 텍스트 강조 | 팝업 애니메이션 |
| 완성 결과물 | 2초 정지 + 살짝 줌인 |
| 빠른 진행 | 2-3배속 + 빠른 전환 |

### BGM 추천
- **장르**: 밝은 코퍼레이트, 테크 느낌
- **볼륨**: 나레이션 대비 20-30%
- **추천 소스**:
  - 유튜브 오디오 라이브러리
  - Artlist (유료)
  - Epidemic Sound (유료)

### 색보정
- **밝기**: +5~10%
- **대비**: 약간 증가
- **채도**: 원본 유지 또는 약간 증가
- **일관성**: 모든 클립 동일 적용
