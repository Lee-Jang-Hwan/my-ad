# 삽가능 스튜디오 서비스 기획서

**문서 버전**: v1.0
**작성일**: 2024년 12월
**서비스 URL**: https://sappstudio.kr

---

## 1. 서비스 개요

### 1) 프로젝트 명
**삽가능 스튜디오 (Sappstudio)**

### 2) 한 줄 소개
> "이미지와 상품명만 있으면 OK! AI가 5분 만에 전문적인 광고 영상을 만들어드립니다"

### 3) 기획 배경 & 문제 정의

#### 시장 현황
- 국내 소상공인 약 **650만 개** 사업체 (전체 사업체의 87%)
- SNS 광고 시장 **연평균 15% 성장** 지속
- 숏폼 콘텐츠(릴스, 쇼츠, 틱톡) 수요 급증

#### 문제 정의

| 문제 | 현황 | 영향 |
|------|------|------|
| **높은 제작 비용** | 영상 외주 시 건당 30~100만 원 | 소상공인 마케팅 예산 부담 |
| **긴 제작 시간** | 평균 1~3일 소요 | 시의성 있는 마케팅 불가 |
| **기술 진입 장벽** | 영상 편집 도구 학습 필요 | 디지털 마케팅 포기 |
| **광고 문구 고민** | 전문 카피라이터 부재 | 효과적인 메시지 전달 실패 |
| **AI 도구 복잡성** | 프롬프트 엔지니어링 필요 | 일반인 활용 어려움 |

#### 기회
- AI 기술 발전으로 **영상 자동 생성** 기술 성숙
- 정부의 **소상공인 디지털 전환** 정책 지원 확대
- 간편한 **크레딧 기반 SaaS 모델** 확산

### 2) 각 기능의 사용 목적 / 사용 시나리오 / 기대 효과

#### 🎬 기능 1: AI 광고 영상 생성 [MUST - 최우선]

| 항목 | 내용 |
|------|------|
| **기능명** | AI 광고 영상 자동 생성 |
| **사용 목적** | 상품 이미지와 정보만으로 전문적인 광고 영상 제작 |
| **소모 크레딧** | 80 크레딧/건 (약 12,000~14,000원) |

**사용 시나리오**:
```
1. 사용자가 "영상 만들기" 메뉴 클릭
2. 상품 이미지 드래그 앤 드롭으로 업로드 (JPG/PNG/WebP, 최대 10MB)
3. 상품명 입력 (예: "프리미엄 무선 이어폰")
4. "광고문구 생성" 버튼 클릭 → AI가 5개 광고 문구 생성 (30초)
5. 마음에 드는 문구 1개 선택
6. "영상 생성" 버튼 클릭 → 크레딧 80 차감
7. 실시간 진행 상황 화면에서 8단계 추적:
   - 초기화 → 이미지 정제 → 영상 생성 → 음성 생성
   → 자막 생성 → 최종 합성 → 완료
8. 3~5분 후 영상 완성 → 미리보기 및 다운로드 (MP4, 세로 9:16)
```

**기대 효과**:
- 영상 제작 비용 95% 절감 (50만 원 → 1.5만 원)
- 제작 시간 99% 단축 (2일 → 5분)
- 전문가 수준 품질로 브랜드 이미지 향상

```
   - 감성적 문구: "당신의 일상에 음악을 더하다"

   - 문제 해결: "끊김 없는 연결, 완벽한 사운드"
   - 신뢰 구축: "10만 고객이 선택한 프리미엄"
   - 행동 유도: "한정 수량, 지금 바로 구매하세요"
4. 사용자가 1개 선택 또는 "다시 생성" 요청
```

**기대 효과**:
- 광고 문구 고민 시간 제거
- 다양한 스타일 제안으로 A/B 테스트 용이
- 전문 카피라이터 수준의 문구 확보

### 3) 우선 순위 (Must / Should / Could)

| 우선순위 | 기능 | 이유 |
|----------|------|------|
| **MUST** | AI 영상 생성 | 핵심 가치 제안, 없으면 서비스 의미 없음 |
| **MUST** | AI 이미지 생성 | 영상 대비 저렴, 진입 장벽 낮춤 |
| **MUST** | AI 문구 생성 | 영상/이미지의 필수 구성요소 |
| **MUST** | 크레딧 충전 | 수익 모델의 핵심 |
| **MUST** | 회원가입/로그인 | 서비스 이용의 기본 |
| **MUST** | 대시보드 | 콘텐츠 관리 필수 |
| **MUST** | 관리자 기능 | 서비스 운영에 필수 |
| **SHOULD** | 프로모션 팝업 | 신규 가입 유도, 이벤트 홍보 |
| **SHOULD** | 공개/비공개 설정 | 샘플 영상 확보용 |
| **SHOULD** | 결제 내역 상세 | 사용자 편의 |
| **COULD** | 프로필 관리 | 부가 기능, 우선순위 낮음 |
| **COULD** | SNS 직접 업로드 | Phase 2에서 고려 |
| **COULD** | 팀 협업 기능 | Phase 3에서 고려 |

#### 화면별 상세 명세

| 화면 | 입력값 | 유효성 검증 | 예외 처리 |
|------|--------|-------------|-----------|
| **이미지 업로드** | 이미지 파일 | - 형식: JPG/PNG/WebP<br>- 크기: 최대 10MB<br>- 해상도: 최소 300x300 | - 형식 오류: "지원하지 않는 형식입니다"<br>- 크기 초과: "10MB 이하만 가능합니다"<br>- 업로드 실패: 재시도 버튼 표시 |
| **상품 정보 입력** | 상품명(필수), 설명(선택), 카테고리(선택) | - 상품명: 1~200자<br>- 설명: 최대 1000자 | - 상품명 미입력: "상품명을 입력해주세요"<br>- 글자수 초과: 실시간 카운터로 제한 |
| **광고문구 선택** | 5개 중 1개 선택 | - 필수 선택 | - 미선택 시: "문구를 선택해주세요"<br>- 생성 실패: "다시 생성" 버튼 활성화 |
| **영상 생성** | 크레딧 차감 동의 | - 잔액 확인 (≥80) | - 크레딧 부족: 충전 모달 표시<br>- 생성 실패: 크레딧 환불 + 재시도 안내 |
| **크레딧 충전** | 요금제 선택, 결제 정보 | - 카드 정보 유효성 | - 결제 실패: 실패 사유 안내 + 재시도<br>- 네트워크 오류: 결제 상태 확인 안내 |
